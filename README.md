# Learning R: Data Analysis & Statistical Computing

A comprehensive learning repository documenting my journey through **R programming** and data analysis concepts. This project showcases foundational and intermediate skills in statistical computing, data manipulation, visualization, and reproducible research using Kaggle workbooks and structured exercises.

## 📌 Overview

This repository contains practical workbooks and scripts built while learning R, with a focus on:
- **Data Manipulation**: Working with vectors, lists, data frames, and tibbles
- **Data Visualization**: Creating publication-quality graphics with ggplot2
- **Statistical Analysis**: Hypothesis testing, regression models, and exploratory data analysis
- **Kaggle Workbooks**: Real-world datasets and competitions to apply theoretical knowledge

**Repository Structure**: Organized by learning progression, each workbook builds on previous concepts.

## 🚀 Quick Start

### Prerequisites
- R 4.0+ ([Download R](https://www.r-project.org/))
- RStudio ([Download RStudio](https://www.rstudio.com/products/rstudio/download/))

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Prasmitprayansu/Learning-R.git
   cd Learning-R
   ```

2. **Install required packages**:
   ```r
   install.packages(c("tidyverse", "ggplot2", "dplyr"))
   ```

3. **Open workbooks**:
   - Open any `.Rmd` file in RStudio
   - Click "Knit" to compile and view outputs

4. **Run scripts**:
   ```r
   source("01-Getting-Started/first_steps.R")
   ```
